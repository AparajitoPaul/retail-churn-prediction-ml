##  File Descriptions

### cleaned_retail.xlsx
Cleaned transaction data after removing:
- Cancelled invoices
- Negative quantities
- Missing customer IDs

---

### historical_window.xlsx
Transactions filtered to pre-cutoff period for feature engineering.

---

### churn_labels.xlsx
Time-based churn labels defined using 90-day forward window.

---

### customer_churn_base.xlsx
Merged customer-level dataset prior to advanced feature engineering.

---

### modeling_dataset.xlsx
Final modeling dataset including:
- RFM features
- Behavioral volatility metrics
- Revenue trend
- Seasonality ratio
- Time-based churn label

This file is used for model training and evaluation.
